An Overview of the Clojure Core API


This is API documentation for the latest committed checkin of the Clojure core.

Important Clojure resources

  • The official source code for clojure is on the Clojure GitHub source page .
  • Issues and other information for clojure are maintained in the Clojure Assembla space .
  • This documentation is maintained in the gh-pages branch of clojure on GitHub and is always available online GitHub pages for Clojure . (Currently stored in a fork on Tom Faulhaber's GitHub account.) If you wish to have a version for off-line use you can use the download button on the gh-pages branch page at GitHub .
  • Issues related to clojure and the various pieces of functionality within it are discussed in the Clojure Google group .
  • Discussions among clojure developers take place in the Clojure Dev Google group .
